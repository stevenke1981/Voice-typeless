# Models

Models are not included in the repository due to their size.
Download them using the built-in model manager (coming soon).

## Supported Models

| Model | Size | Speed | Accuracy | Languages |
|-------|------|-------|----------|-----------|
| SenseVoice Small | ~256 MB | Fast | High | zh, en, ja, ko, yue |
| Whisper Tiny | ~39 MB | Fastest | Medium | 99 languages |
| Custom ONNX | Varies | Varies | Varies | User-defined |

Place model files in this directory or configure a custom path in Settings.
